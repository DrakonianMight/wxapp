# Docker Deployment Guide

## Quick Start

### Using Docker Compose (Recommended)

1. **Build and run:**
   ```bash
   docker-compose up -d
   ```

2. **Access the app:**
   Open http://localhost:8501

3. **View logs:**
   ```bash
   docker-compose logs -f
   ```

4. **Stop the app:**
   ```bash
   docker-compose down
   ```
